Max Integer Solver

Max Integer Solver es una aplicación Spring Boot desarrollada para resolver el problema de encontrar el máximo entero no negativo k tal que 0 ≤ k ≤ n y k mod x = y. Este proyecto implementa una solución algorítmica para el problema descrito en Codeforces - Problem 1374A.
